# 프로젝트 입력 요약

## 앱 설명
시흥시작은도서관협의회 공식 홈페이지. 시흥시 100개+ 작은도서관 중 20여개 소속 도서관들의 소식, 사업, 커뮤니티를 담는 웹사이트.

## 핵심 컨셉
- "도란도란" — 작은도서관들이 모여 이야기를 나누는 따뜻한 커뮤니티
- 동네 사랑방, 주민 소통, 마을 문화 공간
- 2025년 12월 16일 공식 출범

## 주요 기능
1. 메인 페이지: 협의회 소개, 최신 소식, 주요 사업 하이라이트
2. 협의회 소개: 인사말, 설립 목적, 연혁, 조직도, 회칙
3. 회원 도서관: 20여개 소속 도서관 (이름, 위치, 사진, 운영시간, 특색프로그램, 지도)
4. 소식/공지: 공지사항, 활동 소식(사진), 보도자료
5. 사업 안내: 연간 사업 계획, 진행 프로그램, 결과 보고
6. 커뮤니티(도란도란): 자유게시판, 추천도서, 독서모임, 사람도서관
7. 자료실: 회의록, 자료집, 서식 다운로드
8. 갤러리: 활동 사진/영상

## 대상 사용자
- 소속 도서관 운영자/관장
- 시흥시 주민
- 작은도서관 관심 있는 일반인

## 기술 요구사항
- 규모: MVP
- 기술 스택: Next.js + Tailwind + SQLite (Prisma)
- 모바일 반응형 필수
- 관리자 페이지 (게시글, 도서관 정보 관리)
- SEO 최적화
- 한국어 전용
- 배포: Vercel
- 디자인: 따뜻하고 친근한 (녹색/갈색 자연 계열 + 책 모티프)
